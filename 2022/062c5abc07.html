2. X server

2.1. On the Windows side: Xming

I also tried VcXsrv. However, it failed to be launch. Thus I installed xming. After installing, notice that No Access Control may should be picked when start Xlaunch. Then a progress xming server will be started (see it in your task bar).

Windows Side Done.

2.2. On the linux side: $DISPLAY

The DISPLAY variable should be set as :0.0. For bash shell, write export DISPLAY=:0.0 in your .bashrc. For fish shell, write set -x DISPLAY :0.0 in your fish configuration file.

Then you can run a gui program on your linux and get the window on the Windows side.
