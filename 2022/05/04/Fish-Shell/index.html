1. Configuration of shortcuts for fish

Use bind to configure the shortcuts, e.g., bind -M insert \ck 'accept-autosuggestion'. -M is used to define the mode of the shortcut. accept-autosuggestion is the input function, which means accept the current autosuggestion. For more special function see this.

In fish bind -M normal can not work for normal mode of vi input method. However, bind -M default works fine for the normal mode of the vi input method.

2. Set the default editor in fish

fzf allows fish to use fuzzy search. \co is a shortcut of fzf-fish to open a file by fuzzy search with default editor. The default editor can be set by set -gx EDITOR nvim.
